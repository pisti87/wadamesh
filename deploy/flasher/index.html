wada·mesh

Touch-UI MeshCore firmware — install over USB, right from your browser.

Heltec V4 + TFT

ESP32-S3 with the CHSC6x touch display.

Use Chrome or Edge on desktop (Web Serial).

LilyGo T-Deck / Plus

ESP32-S3 keyboard + trackball handheld.

Use Chrome or Edge on desktop (Web Serial).
How it works: plug the device in over USB, click Connect, pick the serial port, and let it flash. This installs the full image (a clean install — it erases the device). On a T-Deck, hold the trackball while tapping reset if it doesn't enter download mode on its own. Works in Chrome / Edge on desktop.
